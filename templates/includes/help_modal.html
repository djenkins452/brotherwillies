Home Dashboard

  • View upcoming games and top value picks at a glance.
  • Edge = difference between our model's probability and the market's implied probability.
  • Positive edge means our model sees more value than the market.
  • Log in to unlock the full Value Board and personal features.
{% elif help_key == 'value_board' %}

Value Board

  • Games ranked by model-vs-market misalignment (edge).
  • House Edge = House model probability minus market probability.
  • User Edge = Your model probability minus market probability.
  • Delta = Your model minus house model (shows where you disagree).
  • Use filters to focus on specific spreads, odds, or minimum edge.
  • Your favorite team is always included when the preference is set.
  • Confidence levels reflect data freshness and completeness.
{% elif help_key == 'cfb_hub' %}

College Football Hub

  • Browse conferences and upcoming games.
  • Tap a conference to see its teams and matchups.
  • Tap a game for detailed analysis with house and user model comparisons.
{% elif help_key == 'game_detail' %}

Game Detail

  • Side-by-side comparison of market, house, and user model probabilities.
  • Line movement arrows show how odds have changed over time.
  • Injury impacts are factored into model calculations.
  • Confidence reflects how recent and complete the underlying data is.
{% elif help_key == 'my_model' %}

My Model

  • Adjust weights to create your own probability model.
  • Rating Weight = how much team strength matters.
  • HFA Weight = home field advantage importance.
  • Injury Weight = how much injuries affect probabilities.
  • Recent Form = emphasis on recent performance.
  • Conference = conference strength adjustment.
  • Your model never overwrites the house model.
{% elif help_key == 'my_stats' %}

My Statistics

  • Track your analysis activity and model comparison history.
  • See how often your model agrees with the house model.
  • Review confidence distribution of games you've analyzed.
  • Past performance does not predict future outcomes.
{% elif help_key == 'performance' %}

Model Performance

  • Accuracy = how often the model correctly predicted the winner.
  • Calibration = how well predicted probabilities match actual outcomes.
  • CLV = Closing Line Value, comparing your analysis timing to final market odds.
  • These are model quality indicators, not financial results.
{% elif help_key == 'parlays' %}

Parlay Analytics

  • Build parlays to analyze combined probabilities.
  • Correlation risk flags when legs from the same game may be correlated.
  • High correlation applies a conservative haircut to probability estimates.
  • This is analysis only - not betting advice.
